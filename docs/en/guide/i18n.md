# Internationalization

MinAd UI provides comprehensive internationalization support, allowing you to easily adapt the component library to different language environments.

## Quick Start

### Configure Language

You can use the `MdConfigProvider` component to configure the language globally:

```vue
<template>
  <md-config-provider locale="en">
    <app />
  </md-config-provider>
</template>

<script setup lang="ts">
import { MdConfigProvider } from 'minad-ui'
</script>
```

### Supported Languages

- Chinese (zh-cn)
- English (en)

## Manually Set Language

You can also manually set the language using the `setLocale` method:

```ts
import { createI18n } from 'minad-ui/src/i18n/i18n'

const i18n = createI18n()
i18n.setLocale('zh-cn')
```

## Custom Language Pack

You can add custom language packs using the `mergeLocaleMessage` method:

```ts
import { createI18n } from 'minad-ui/src/i18n/i18n'

const i18n = createI18n()

// Add custom language pack
i18n.mergeLocaleMessage('ja', {
  input: {
    placeholder: 'Please input'
  },
  button: {
    confirm: 'Confirm'
  },
  form: {
    required: 'This field is required'
  }
})

// Use custom language
i18n.setLocale('ja')
```

## Usage in Components

Inside components, you can use the `useI18n` composable function to get the translation function:

```vue
<template>
  <div>{{ t('hello') }}</div>
</template>

<script setup lang="ts">
import { useI18n } from 'minad-ui/src/i18n/i18n'

const { t } = useI18n()
</script>
```

## Translation Function

### t(key, options?)

- `key`: Translation key name
- `options`: Optional parameters for replacing variables in translation texts

Example:

```vue
<template>
  <div>{{ t('form.minLength', { min: 5 }) }}</div>
</template>
```

## Translation Keys

### Input Component

- `input.placeholder`: Input box default placeholder

### Button Component

- `button.confirm`: Confirm button
- `button.cancel`: Cancel button
- `button.submit`: Submit button
- `button.reset`: Reset button

### Form Component

- `form.required`: Required field prompt
- `form.minLength`: Minimum length prompt
- `form.maxLength`: Maximum length prompt
