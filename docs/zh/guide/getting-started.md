# Getting Started

欢迎使用 Md UI！这是一个基于 Vue 3 的移动端 UI 组件库。

## 简介

Md UI 是专为移动端设计的 UI 组件库，提供了一系列常用的组件，帮助开发者快速构建高质量的移动端应用。

## 特点

- **Vue 3 驱动**：基于 Vue 3 的 Composition API 和响应式系统
- **移动端优先**：专为移动端设计和优化
- **TypeScript 支持**：完整的 TypeScript 类型定义
- **自定义主题**：支持自定义主题和样式变量
- **组件丰富**：提供常用的移动端 UI 组件
- **易用性高**：简洁的 API 设计，易于上手
