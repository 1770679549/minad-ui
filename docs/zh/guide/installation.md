# Installation

## 环境要求

- Vue 3.0+ 
- TypeScript 4.5+ (可选)

## 安装

### 使用 npm

```bash
npm install minad-ui --save
```

### 使用 yarn

```bash
yarn add minad-ui
```

### 使用 pnpm

```bash
pnpm add minad-ui
```

## 版本说明

- **最新稳定版**：1.0.0
- **支持的 Vue 版本**：Vue 3.0+
- **浏览器支持**：iOS 12+, Android 6+
