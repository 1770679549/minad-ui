# 国际化

MinAd UI 提供了全面的国际化支持，可以轻松地将组件库适配到不同的语言环境。

## 快速开始

### 配置语言

您可以使用 `MdConfigProvider` 组件来全局配置语言：

```vue
<template>
  <md-config-provider locale="zh-cn">
    <app />
  </md-config-provider>
</template>

<script setup lang="ts">
import { MdConfigProvider } from 'minad-ui'
</script>
```

### 支持的语言

- 中文 (zh-cn)
- 英文 (en)

## 手动设置语言

您也可以通过 `setLocale` 方法手动设置语言：

```ts
import { createI18n } from 'minad-ui/src/i18n/i18n'

const i18n = createI18n()
i18n.setLocale('en')
```

## 自定义语言包

您可以通过 `mergeLocaleMessage` 方法添加自定义语言包：

```ts
import { createI18n } from 'minad-ui/src/i18n/i18n'

const i18n = createI18n()

// 添加自定义语言包
i18n.mergeLocaleMessage('ja', {
  input: {
    placeholder: '入力してください'
  },
  button: {
    confirm: '確認'
  },
  form: {
    required: 'このフィールドは必須です'
  }
})

// 使用自定义语言
i18n.setLocale('ja')
```

## 组件内使用

在组件内部，您可以使用 `useI18n` 组合式函数来获取翻译函数：

```vue
<template>
  <div>{{ t('hello') }}</div>
</template>

<script setup lang="ts">
import { useI18n } from 'minad-ui/src/i18n/i18n'

const { t } = useI18n()
</script>
```

## 翻译函数

### t(key, options?)

- `key`: 翻译键名
- `options`: 可选参数，用于替换翻译文本中的变量

示例：

```vue
<template>
  <div>{{ t('form.minLength', { min: 5 }) }}</div>
</template>
```

## 翻译键名

### Input 组件

- `input.placeholder`: 输入框默认占位符

### Button 组件

- `button.confirm`: 确认按钮
- `button.cancel`: 取消按钮
- `button.submit`: 提交按钮
- `button.reset`: 重置按钮

### Form 组件

- `form.required`: 必填项提示
- `form.minLength`: 最小长度提示
- `form.maxLength`: 最大长度提示
